# Revision Latency v0.2

Revision Latency measures how many narrative units it takes for a text to move from a destabilized loop state into adaptive processing.

## State Groups

Destabilized:
- Overloaded
- Fragmented
- Rigid
- Defensive

Adaptive:
- Flexible
- Integrated

Neutral:
- Stable/Neutral
- Mixed / Review Needed

## Metrics

- Latency to Flexible
- Latency to Integrated
- Stalled State Count
- No-Resolution Flag
- Regression Count
- Final State
- Recovery Arc
- Recovery Stability
- Trajectory Score
