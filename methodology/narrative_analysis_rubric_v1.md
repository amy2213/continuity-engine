# Narrative Analysis Rubric v1

The rubric scores text across eight dimensions and assigns a loop-state classification.

Core thesis: Identity is continuity under revision.

Dimensions:
1. Self-Reference Density
2. Identity Fixity
3. Prediction / Expectation Language
4. Threat Weighting
5. Feedback Integration
6. Narrative Flexibility
7. Fragmentation / Contradiction
8. Updated Continuity

Loop states:
- Flexible
- Integrated
- Defensive
- Rigid
- Fragmented
- Overloaded
- Stable/Neutral
- Mixed / Review Needed
